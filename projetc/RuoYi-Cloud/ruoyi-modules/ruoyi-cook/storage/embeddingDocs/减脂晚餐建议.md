# 减脂晚餐建议

## 适用人群
适合正在控制体重、希望晚餐低热量但有饱腹感的人群，也适合久坐、下班较晚但仍需要规律进食的人。

## 核心建议
减脂晚餐优先采用“高蛋白 + 高纤维蔬菜 + 少量优质碳水”的结构。蛋白质可以选择鸡胸肉、鱼虾、鸡蛋、豆腐或低脂奶制品；蔬菜选择绿叶菜、番茄、菌菇、黄瓜、西兰花；主食控制在半碗杂粮饭、一个小红薯或少量玉米。

烹调方式以蒸、煮、炖、少油快炒为主，避免油炸、浓油赤酱和大量沙拉酱。晚餐不要完全不吃主食，长期过低碳水容易导致饥饿、暴食和运动恢复变差。

## 禁忌/风险
不要只吃水果、只喝代餐或完全不吃晚餐。胃病、糖尿病、孕期和慢性病人群需要结合医生建议调整总热量和碳水比例。

## 常见问答
问：减脂晚餐吃什么比较好？
答：可以吃一份清蒸鱼或鸡胸肉，搭配两拳蔬菜和半拳杂粮主食。重点是少油、足量蛋白、蔬菜占盘子一半。

问：晚上想减肥怎么吃？
答：不要靠饿，可以把晚餐提前到睡前 3 小时左右，选择豆腐蔬菜汤、虾仁西兰花、鸡蛋番茄和少量粗粮。
