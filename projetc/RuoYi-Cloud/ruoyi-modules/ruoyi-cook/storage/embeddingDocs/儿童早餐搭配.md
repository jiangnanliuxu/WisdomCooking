# 儿童早餐搭配

## 适用人群
适合学龄前儿童、小学生和需要改善早餐质量的家庭参考。

## 核心建议
儿童早餐应包含优质蛋白、主食、蔬果和适量奶制品。常见组合包括鸡蛋 + 全麦面包 + 牛奶 + 水果，或者杂粮粥 + 鸡蛋 + 小菜 + 酸奶。早餐要保证能量和营养密度，避免只吃甜面包、饼干、含糖饮料或油炸食品。

蛋白质可以来自鸡蛋、牛奶、酸奶、瘦肉、鱼、豆腐。主食可以选择包子、馒头、面条、燕麦、玉米、杂粮粥。水果不替代正餐，果汁也不等于水果。

## 禁忌/风险
儿童咀嚼和吞咽能力有限，坚果、整颗葡萄等食物要注意窒息风险。过敏儿童应避开已知过敏原。

## 常见问答
问：小学生早餐怎么搭配？
答：建议一份主食、一份蛋白、一份奶或豆制品，再加少量水果或蔬菜。例如鸡蛋牛奶配全麦三明治和苹果片。

问：孩子早上没胃口怎么办？
答：可以把早餐做小份、温热、易入口，前一晚避免过晚吃夜宵。

## 关键词
儿童早餐、小学生早餐、蛋白质、碳水、牛奶、鸡蛋、避免高糖
