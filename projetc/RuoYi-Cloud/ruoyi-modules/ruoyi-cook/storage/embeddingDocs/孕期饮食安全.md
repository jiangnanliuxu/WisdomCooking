# 孕期饮食安全

## 适用人群
适合孕期、备孕和关注孕期营养安全的人群参考。

## 核心建议
孕期饮食要重视叶酸、铁、钙、优质蛋白和安全卫生。可以选择熟透的肉蛋奶、鱼虾、豆制品、绿叶菜、全谷物和水果。食物要充分加热，减少生冷、半熟和来源不明食品。

孕期应避免生鱼片、未熟蛋、未消毒奶、酒精和高汞鱼类。咖啡因需要控制总量。外食时注意食材新鲜和彻底加热，剩菜保存和复热要严格。

## 禁忌/风险
孕期个体差异大，妊娠糖尿病、高血压、贫血或体重增长异常应遵医嘱。任何保健品和草药补充剂都不宜自行大量使用。

## 常见问答
问：孕妇能不能吃生冷食物？
答：不建议吃生鱼片、半熟蛋、生冷海鲜和未充分清洗的冷食。关键风险是细菌、寄生虫和食源性疾病。

问：孕期怎么补铁？
答：可以吃瘦肉、动物血、深绿色蔬菜，并搭配富含维生素 C 的食物促进吸收。
