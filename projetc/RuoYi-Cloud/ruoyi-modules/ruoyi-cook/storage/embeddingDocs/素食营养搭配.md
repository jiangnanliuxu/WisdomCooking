# 素食营养搭配

## 适用人群
适合素食者、减少肉类摄入、关注植物蛋白和营养均衡的人群。

## 核心建议
素食饮食要重视蛋白质、铁、锌、钙、维生素 B12 和 Omega-3。蛋白质可以来自豆腐、豆浆、豆干、黄豆、黑豆、鹰嘴豆、扁豆、坚果和全谷物。谷物和豆类搭配可以改善氨基酸互补。

纯素人群尤其要关注维生素 B12，通常需要强化食品或补充剂。铁来源包括豆类、深绿色蔬菜、芝麻酱，并搭配水果或维生素 C 丰富食物促进吸收。

## 禁忌/风险
儿童、孕期、哺乳期和老年素食者需要更严谨的营养规划。只吃蔬菜水果不是均衡素食。

## 常见问答
问：素食怎么保证蛋白质够？
答：每餐安排豆制品或豆类，搭配全谷物、坚果和奶蛋类。如果是纯素，应特别关注 B12。

问：素食容易缺什么？
答：可能缺 B12、铁、锌、钙和优质蛋白，需要通过搭配或补充剂解决。
