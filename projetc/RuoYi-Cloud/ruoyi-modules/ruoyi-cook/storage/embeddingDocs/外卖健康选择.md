# 外卖健康选择

## 适用人群
适合工作日经常点外卖、想控制油盐糖和总热量的人群。

## 核心建议
点外卖优先选择清蒸、炖煮、白灼、少油炒的菜，搭配蔬菜、蛋白质和适量主食。备注少油、少盐、酱料分装。避免长期选择炸鸡、重辣火锅、麻辣香锅、奶茶和高糖甜品。

健康组合可以是米饭半份 + 清蒸鱼或鸡肉 + 两份蔬菜，也可以选择轻食但注意酱料和主食量。外卖汤汁常含油盐，少拌饭少喝汤。

## 禁忌/风险
肾病、高血压、糖尿病和痛风人群点外卖更要注意盐、糖、嘌呤和脂肪。外卖不能替代长期均衡饮食。

## 常见问答
问：点外卖怎么吃健康一点？
答：选少油少盐的套餐，保证有蛋白和蔬菜，主食半份，酱料另放，少喝汤汁，少点含糖饮料。

问：轻食一定健康吗？
答：不一定。沙拉酱、油醋汁、培根和过少主食都可能不适合你的目标。
